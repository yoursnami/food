Prototype Overview

An extensive user interface is built into this storage box to help with simultaneous synchronization with the system, allowing you to track it from a distance using multiple devices logged in under the same family. This tracking is designed to assist users from various backgrounds in monitoring their stored food status, which is mostly identified using the expiry date label on food packages. Apart from keeping you safe from consuming food that is no longer safe, this feature, with various functional purposes such as inventory, groceries, and recipes, helps users explore useful actions. Prioritizing user-friendliness and elderly users, some actions are equipped with voice commands and visually clear instructions to ease the learning process of the system. This prototype is made using a website, with sequential actions set for each tab and empty boxes symbolizing the food storage box. We are still in the process of developing the actual synchronized device and interface. Please consider the current output as a genuine reflection of the design, and feel free to point out anything that may not work as expected. Let us know if you experience any inconvenience during this testing, so we can further improve the design.

4.0 SCENARIO TASKS (INDEX CARDS)

  • Task 1: Place your face within the face recognition circle displayed on the screen and choose your username.
  • Task 2: Add a new food item to storage by either scanning the food barcode or entering the details manually.
  • Task 3: Point your back camera at the barcode so it can be captured and automatically update the expiry date into the system.
  • Task 4: Enter the food name, select the appropriate food category, and pick the expiry date using the digital calendar on the screen.
  • Task 5: Monitor the inventory to check the food status—whether it is nearly expired or already expired.
  • Task 6: Choose an expiry reminder action where you can decide to consume, discard, or snooze the selected item.
  • Task 7: Based on the available items in inventory, suggested recipes will be displayed. Choose one of the suggested recipes to be directed to the recipe page.
  • Task 8: Select any item you identify as out of stock so it will be automatically added to your grocery list.
  • Task 9: Tap on another user’s activity log to view the activity of other users from the same family account.
