Smart Food Storage Box

Smarter Storage, Fresher Food

Project Overview

The Smart Food Storage Box is a smart and energy-efficient solution aimed at reducing food waste and improving storage management. Our box uses IoT sensors to monitor temperature, humidity, and food shelf life, notifying users via a mobile app when conditions are not optimal or when food is nearing expiration.

Our early sketch outlining the concept, features, and user scenarios.

Our Proposal

This website and project are part of our proposal for the Human Interaction course, where we aim to create a user-friendly, tech-enabled solution that solves a real-world problem. The Smart Food Storage Box focuses on enhancing the interaction between humans and technology to promote sustainable food practices.

Key Features

  • Real-time temperature and humidity monitoring
  • Mobile app notifications for food expiration
  • Energy-efficient cooling system
  • Remote control and data sync via mobile application

Technology Stack

Hardware: IoT sensors, Microcontroller (ESP32)
Software: React Native App, Node.js Backend, Firebase database
Connectivity: Wi-Fi module for real-time sync
